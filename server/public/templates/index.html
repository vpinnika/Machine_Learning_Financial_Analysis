Introduction

This project is makes use of machine learning algorithms and technical indicators to perform financial analysis on the stock market. Nothing can predict the stock market to a tee, especially given the volatile market conditions. However, we can use tools to attempt to forecast directional movements in the market. This app is built on several technologies including Scikit-Learn, Flask and Pandas.
